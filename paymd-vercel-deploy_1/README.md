# pay.md — Telegram Mini App

QR-меню и онлайн-оплата для Woloshin Banya.

## Деплой

1. Загрузи этот репозиторий на GitHub
2. Подключи к Vercel (vercel.com)
3. Добавь ссылку в BotFather → Menu Button

## Структура

- `index.html` — Telegram Mini App (меню + корзина + оплата + вызов официанта)
- `vercel.json` — конфиг для Vercel
